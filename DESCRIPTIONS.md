# Descriptions des fichiers

## Captures d’écran de configuration et d’administration

- `Capture d'écran 2026-05-07 173040.png` : capture d’écran illustrant une étape de configuration ou d’administration du serveur Linux.
- `Capture d'écran 2026-05-08 115824.png` : capture d’écran montrant une autre étape de configuration ou de vérification sur le serveur Linux.
- `Capture d'écran 2026-05-10 234703.png` : capture d’écran liée à une manipulation, un test ou une validation de configuration du serveur Linux.

## Gestion des accès et des services

- `config access.png` : image montrant un écran de configuration d’accès ou de permissions sur le serveur.
- `srvsamba.png` : capture d’écran de la configuration ou de l’état du service Samba sur le serveur Linux.

## Fichiers de configuration

- `serverli config.txt` : fichier texte contenant des éléments de configuration du serveur Linux.
